Selektor Universal CSS

Selektor Universal merupakan selektor yang digunakan untuk menyeleksi semua elemen pada jangkauan (scope) tertentu. Selektor Universal biasanya digunakan untuk me-reset CSS. Pada halaman HTML, ada beberapa CSS bawaan browser seperti padding dan margin pada elemen tertent. Reset bertujuan untuk menghilangkan padding dan margin tersebut.

Contoh Selektor Universal pada HTML ini, berupa pemberian style pada semua elemen agar memiliki garis solid dengan ukuran 1px berwarna grey. Serta memberikan margin sebersar 10px dan padding seberar 30px pada setiap elemen HTML ini.
